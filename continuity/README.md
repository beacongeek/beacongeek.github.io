# THE CONTINUITY ARCHIVE

A fully static Github Pages site dedicated to the translation and analysis
of the Continuity Manuscript.

Features:
- Pure HTML/CSS/JS
- Markdown powered
- Sidebar navigation
- Dynamic page indexing
- Github Pages compatible
- Manuscript always visible
- Human vs AI analysis sections
- Ancient techno-occult aesthetic

## DEPLOYMENT

1. Create a Github repo
2. Upload all files
3. Enable Github Pages
4. Set source branch to `main`
5. Done

---

Install marked.js:

Download:
https://cdn.jsdelivr.net/npm/marked/marked.min.js

Save to:
`/libs/marked.min.js`

---

Add your manuscript image to:

`/manuscript/manuscript.png`

---

Add markdown pages inside:

`/content/humans/`
`/content/ai/`
